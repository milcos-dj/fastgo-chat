# Fastgo-Chat

支持功能：

- 会话功能
- 本地记录持久化
- 代码复制
- 代码行号
- 代码高亮
- Markdown显示
- 自定义上下文数量
- 自定义接入点
- Azure 接入点
- 自定义模型
- 自定义 Prompt
- 长文本编辑
- 响应式，跨平台支持
